// UI ELEMENTS
05 — Usage Guide

Arc

Primary actions, live indicators, links. The brightest thing on screen — used with intention, never decoration.

Navy

Secondary actions. Sits below Arc in the hierarchy — the same cold water family, just deeper.

Slate

Muted tags, secondary icons, anything that supports without competing. The lid, not the arc.

Latch

The maroon-black near-dark. Use for charcoal UI chrome, deep icon fills, the heaviest surface before true black.

Band

The cool grey separator. Dividers, placeholders, disabled states — quiet but structural.

Cream

Warm accent. Featured labels, notification badges, input focus rings. The warmth against the cold arc.
